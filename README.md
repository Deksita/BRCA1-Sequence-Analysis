# Comparative Analysis of BRCA1 Across Species

OVERVIEW

This project explores the BRCA1 gene across different species to understand how it has been conserved through evolution. BRCA1 is an important gene involved in DNA repair, and mutations in it are linked to breast and ovarian cancer.

Using bioinformatics tools, both DNA and protein sequences were compared to identify conserved regions and study evolutionary relationships.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

OBJECTIVE

To compare BRCA1 gene sequences across selected vertebrates and identify conserved regions, functional domains, and evolutionary patterns.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

TOOLS USED

NCBI (sequence retrieval)

BLAST (similarity search)

Clustal Omega (multiple sequence alignment)

UniProt (protein domain analysis)

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

SPECIES INCLUDED

Human (Homo sapiens)

Chimpanzee (Pan troglodytes)

Cow (Bos taurus)

Rat (Rattus norvegicus)

Mouse (Mus musculus)

Chicken (Gallus gallus)

Frog (Xenopus laevis)

These were chosen to represent different evolutionary distances.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

METHODOOGY

Retrieved BRCA1 DNA and protein sequences from NCBI and verified protein sequences using database records

Performed multiple sequence alignment (MSA) separately for:

DNA sequences

Protein sequences

Analysed conserved regions and variations

Ran BLAST using human BRCA1

Identified functional domains using UniProt

Constructed a phylogenetic tree

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

KEY FINDINGS

Protein sequences showed higher conservation than DNA sequences

Conserved regions matched important functional domains (RING and BRCT)

BLAST confirmed chimpanzee as the closest match to humans

Phylogenetic analysis reflected expected evolutionary relationships

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

WHAT THIS MEANS

Important regions of the BRCA1 protein are preserved across species because they are essential for function.

Even though DNA changes over time, the protein structure remains more stable due to evolutionary pressure.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

LIMITATIONS

Limited number of species analysed

No experimental validation

Results depend on database accuracy

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

CONCLUSION

This project shows how simple bioinformatics tools can be used to study gene conservation and evolution. It highlights the importance of BRCA1 and demonstrates how conserved regions often point to critical biological functions.
